# Snack Functions
This is the 15th exercise I have completed as part of the web development master. It focuses on practicing logic, problem-solving, and clean code principles through the definition and invocation of custom JavaScript functions.
Repository name: `js-snacks-functions`

## 📝 Task
Solve the assigned coding challenges (snacks) included in the repository by correctly defining and invoking dedicated functions.

### 🌟 Bonus
* Solve the challenges using both traditional JavaScript functions and modern Arrow Functions.
* Complete the fourth bonus challenge located inside the `05 - bonus` directory.
